
# (Sprint) Planning Poker - GoTH STACK test

#### This is a small test app to test the GoTH stack ([Go](https://go.dev/), [Templ](https://github.com/a-h/templ) + [Htmx](https://htmx.org)), [Gin Web Framework](https://gin-gonic.com/), ([Gorilla](https://pkg.go.dev/github.com/gorilla/websocket@v1.5.3)) Websockets and [tailwind](https://tailwindcss.com).
- !! Warning !! do not use in production; Security features (Like Websocket CheckOrigin, [Gin](https://gin-gonic.com/) Production Mode) not implemented!


### Development

Recommendations:
- Templ ([install](https://templ.guide/quick-start/installation)) run: ```templ generate --watch```
- Air ([install](https://github.com/air-verse/air)) run ```air```
